# UniGrok public operational contract

UniGrok exposes one workspace-neutral Grok harness over Streamable HTTP MCP.

## Start here

- MCP: `/mcp`
- Health: `/healthz`
- Readiness: `/readyz`
- Runtime: `/runtimez`
- Benchmarks: `/benchmarkz`
- Control Center: `/ui/`
- WebMCP discovery: `/.well-known/webmcp`

Use `agent` with only `{"task":"Your request"}`. The live-discovered lead selects direct
work or writes a bounded specialist brief. The gateway validates every selection against
live provider catalogs. Workspace files, Git, the host shell, IDE secrets, and external
MCP servers are never attached.

This surface describes local Compose. In hosted mode, Control OAuth protects MCP and
non-public operational routes, the CLI plane is disabled, and state is instance-local.

Benchmark receipts store route metadata, latency, cost, caller labels, verified outcomes,
fallback categories, and circuit-breaker state. They never store prompt text.
