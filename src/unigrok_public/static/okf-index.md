# UniGrok public operational contract

UniGrok exposes one workspace-neutral Grok harness over Streamable HTTP MCP.

## Start here

- MCP: `/mcp`
- Health: `/healthz`
- Readiness: `/readyz`
- Runtime: `/runtimez`
- Benchmarks: `/benchmarkz`
- Control Center: `/ui/`
- WebMCP discovery: `/.well-known/webmcp`

Use `agent` with only `{"task":"Your request"}`. Grok 4.5 selects direct work or writes
a bounded specialist brief. The gateway validates every selection against live provider
catalogs. Workspace files, Git, the host shell, IDE secrets, and external MCP servers are
never attached.

Benchmark receipts store route metadata, latency, cost, caller labels, verified outcomes,
fallback categories, and circuit-breaker state. They never store prompt text.
