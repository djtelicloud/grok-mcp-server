# Code of Conduct

## Our Standard

UniGrok MCP is a technical collaboration space. Participants are expected to be
professional, direct, and respectful.

Acceptable behavior includes:

- focusing feedback on code, design, evidence, and user impact;
- assuming good intent while still addressing concrete risks;
- welcoming contributors with different experience levels;
- keeping credentials, private data, and sensitive logs out of public threads.

Unacceptable behavior includes:

- harassment, threats, or personal attacks;
- discriminatory language or conduct;
- publishing another person's private information;
- deliberately derailing technical discussion;
- submitting secrets, malware, or intentionally unsafe code.

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or
block participants who violate this code of conduct.

If you need to report a problem, contact the repository maintainers through the
GitHub issue or security channels available on the project.
