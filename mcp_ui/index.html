Get the gateway running

Copy and run this in a terminal. You need Git, Docker Desktop, and uv — nothing else.

git clone https://github.com/djtelicloud/grok-mcp-server.git
cd grok-mcp-server
uv run python main.py init
docker compose up --build -d
Already installed? Run docker compose up --build -d in the project folder, then Recheck.
